LIFEGUARD_MIPT PROJECT
Это проект, сделанный в рамках курса "Практикум по цифровому производству" тремя студентами МФТИ: Колодезным Иваном, Карелиным Данилой, Малышевым Вадимом.
Суть проекта: браслет отслеживает наличие движения и сердцебиения пользователя, и в случае их отсутствия уведомляет об этом экстренные контакты через телеграм бота.
На данном репозитории хранится программный код проекта (использовалась ESP32, программировалась с помощью PlatformIO), а также файлы 3D-моделей корпуса. С принципиальной схемой и разводкой печатной платы можно ознакомиться по ссылке https://easyeda.com/editor#project_id=9dc9597eaed445deb9155b8b403aea16
